[TypeScript Twoslash](https://www.typescriptlang.org/dev/twoslash/) is a pre-processor that enhances your code-samples. It is a markup language for JavaScript and TypeScript.

Expressive Code TwoSlash is a tool for [Expressive Code](https://expressive-code.com/) that allows you to create interactive code examples with TypeScript. It leverages the power of TwoSlash to provide a rich, interactive experience for users, making it easier to understand and learn from code snippets.

```ts twoslash title="example.ts"
// This is an example of a TypeScript code snippet
function greet(name: string) {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```
